Technical Support

1. How do I download files from the Marketplace?

To download a file from our Marketplace, follow these steps:

  • Find the model you want.
  • Click Download Model.
  • Enter the 2 numbers our site provides to verify you are human.
  • Click the Download ZIP or normal file option, depending on how the model was uploaded.
2. What should I do if a download is corrupted or won’t open?

If your downloaded model is corrupt, contains malware, or won’t open:

  • Immediately delete the file from your Mac or Windows computer to prevent any potential harm.
  • Go to create a support ticket.
  • Be sure to include the model’s profile link so our team can review and address the issue promptly.
3. Which software do I need to open 3D model files?

Any software that supports opening the following 3D print file types is compatible for use:

  • .STL
  • .OBJ
  • .3MF
  • .PLY
  • .FBX

If you don’t have software to open these files, popular options include Ultimaker Cura, PrusaSlicer, MeshLab, and Bambu Studio — many of which are free to download.

4. How do I report a bug or technical issue on the website?

To report a technical issue, please create a support ticket.

Our team will review your report and reach out to help resolve the issue you’re experiencing.

5. Is RekietaLabs compatible with mobile devices?

Yes. RekietaLabs is compatible with mobile devices.

However, if you plan to download 3D print files, we recommend doing so on a Mac or Windows computer for the best experience.
